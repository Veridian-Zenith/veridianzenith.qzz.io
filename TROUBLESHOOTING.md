# Build vs Dev Troubleshooting Plan

## Issue
- `npm run dev` works perfectly
- `npm run build` results in blank grey page

## Investigation Steps
- [ ] 1. Examine project configuration files
- [ ] 2. Check build output and structure
- [ ] 3. Analyze potential routing issues
- [ ] 4. Verify asset path configuration
- [ ] 5. Check for JavaScript errors
- [ ] 6. Test built files in isolation
- [ ] 7. Compare dev vs build behavior
- [ ] 8. Implement fixes
- [ ] 9. Verify solution

## Common Causes
1. Base URL/Path configuration
2. Routing strategy (hash vs history)
3. Environment variables missing in production
4. Build configuration issues
5. Asset path references
6. JavaScript/CSS loading errors
